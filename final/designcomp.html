Visual Design

I want to improve the overall visual design of my madlibs project. To accomplish this, I plan to work on theming the form page better and work on improving the storytelling aspect of it. On my forms page, the input boxes are sequenced in one large group. To make the form more digestable, I plan to split up the form into sections where the user can click on a 'next' button that leads to a scroll effect to more questions. I also want to add more imagery to the homepage because I feel like it will help differentiate the content from the results page. I feel that this will all help in building anticipation for the results generated.
